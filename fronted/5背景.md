# 背景background
## 定义和用法
- background简写属性在一个声明中设置所有的背景属性
- 可以设置如下属性：
- background-color
- background-position
- background-size
- background-repeat
- background-origin
- background-clip
- background-attachment
- background-image
- 如果不设置其中某个值，也不会出问题，比如background:#ff0000 url('smiley.gif');也是允许的；
## 设置背景图片
- 需要一个url地址作为参数，例子：background-image:url（图片的路径）
```html
<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title></title>
        <style>
            #box1{
                width:800px;
                height:300px;
                /*设置背景颜色的4种方式*/
                background-color:red;
                background-color:#e00b00;
                background-color:rgb(255,0,0);
                background-color:rgba(255,0,0,.9);/*后面的.9是透明度，1是完全不透明0是完全透明*/
                /*当然这个设置透明度的属性可以单独拿出来设置*/
                opacity:.5;
                margin:0 auto;
                /*设置背景图片的方式*/
                background-image:url(./image/IE盒子.png);/*一般只写这一行它背景不足够大的时候会默认平铺整个界面*/
                background-repeat:repeat;/*向水平方向和垂直方向都进行平铺*/
                background-repeat:repeat-x;/*向水平方向进行平铺*/
                background-repeat:repeat-y;/*向垂直方向进行平铺*/
                background-repeat:no-repeat;/*不进行平铺*/
                /*设置图片位置*/
                background-position:0px 0px;/*background-position:x轴偏移量 y轴偏移量 */
                background-position:center center;
                /*规定背景图像的尺寸*/
                /*length(设置背景图像的宽高，第一个值是宽第二个是高；如果只设置一个值则第二个值会被设置为auto)*/
                /*percentage(以父元素的百分比来设置背景图像的宽高，第一个值是宽第二个值是高；如果只设置一个值第二个值会被设置为auto)*/
                /*cover（把背景图像扩展至足够大，以使背景图像完全覆盖背景区域；背景图像的某些部分也许没法显示在背景定位区域中）*/
                /*contain（把图像扩展至最大尺寸，以使其宽度和高度完全适应内容区域,可能会有留白不会失真）*/
                background-size:400px 200px;
                background-size:100% 100%;
                background-size:cover;
                background-size:contain;
                /*设置背景是否随页面滚动*/
                /*可选值scroll(默认值，背景图片会随页面一起滚动)；*/
                /*fixed（背景图片不随页面滚动，会固定在页面的指定位置；设置该属性的背景，则背景会永远相对于浏览器窗口进行定位；一般这种背景都会设置给body）*/
                background-attachment:scroll;
                background-attachment:fixed;
                /*以上所有都可以在一个设置中设置*/
                background:rgba(255,0,0,.9) url(./image/IE盒子.png) no-repeat center center 100% 100%  cover fixed;
            }
        </style>
    </head>
```

## 使用注意
- 如果背景图片大于元素，默认会显示图片的左上角
- 如果背景图片和元素一样大，则会将背景图片全部显示
- 如果背景图片小于元素大小，则会默认将背景图片平铺以充满元素
## 可以同时
可以同时为一个元素指定背景颜色和背景，这样背景颜色会作为图片的底色，一般情况下，设置背景图片的时候都会设置一下背景颜色
